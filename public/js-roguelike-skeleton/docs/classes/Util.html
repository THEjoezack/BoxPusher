DIRECTIONS_OPPOSITES

Object final static

Defined in src/util.js:89

Maps direction names their opposite direction name.

Example:

{
   up:           {x:  0, y: -1},
   up_right:     {x:  1, y: -1},
   right:        {x:  1, y:  0},
   down_right:   {x:  1, y:  1},
   down:         {x:  0, y:  1},
   down_left:    {x: -1, y:  1},
   left:         {x: -1, y:  0},
   up_left:      {x: -1, y: -1}
}
